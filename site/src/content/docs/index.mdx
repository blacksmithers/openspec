## What is SpecForge Format?

SpecForge Format (SFF) is a versioned, tool-agnostic format for describing software projects as hierarchical specifications that AI agents can execute. It defines how to decompose a project into specifications, epics, and tickets -- a dependency-aware execution graph any compliant agent can consume.

**OpenAPI defined how to describe REST APIs. SpecForge Format defines how to describe software work for agents.**

<CardGrid>
  <Card title="Three Formats" icon="document">

## Quick Install

```bash
npm install -g @specforge/validator

specforge-validate ./my-spec.sf.json   # Valid SpecForge spec (v1.0)
specforge-validate ./my-spec.sf.toon   # Valid SpecForge spec (v1.0) -- toon
```

## Schema URL

```
https://schema.specforge.tech/schema/v1.0/specforge-schema.json
```
